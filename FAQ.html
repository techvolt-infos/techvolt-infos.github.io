Détection des capteurs

Capteurs de température

CAUSES :

Diverses causes peuvent influencer la détection des capteurs donc il est important de déterminer le problème.

Si vous souhaitez démarrer une manipualtion mais que l'un des capteurs dans la section "Port" pour les capteurs de température affiche "N/A" il se peut que vos drivers ne soient pas à jour ou du moins que les capteurs soient mal configurés.

AVERTISSEMENT : 

Avant toute manipulation prenait la précaution de débrancher tout autres capteurs afin de pouvoir faire la correction correctement.


CORRECTION :

Pour commencer veuiller cliquer sur le logo windows en bas à gauche de ordinateur et taper "Gestionnaire de périphériques". Une fois lancer vérifier dans la section "Ports (COM et LPT)" si des composants sont détecter. 

Si rien n'est affiché cela peut-être dû au port USB mal branché ou à votre matériel qui est défectueux.

Dans le cas contraire si vous voyez un composant du type "Périphérique série USB (COM X)" détecter c'est que votre composant utilise le mauvais driver.
Pour remédier à cela faites clique droit sur le composant et sélectionner "Mettre à jour le pilote", ensuite sélectionner "Parcourir mon poste de travail pour rechercher des pilotes", cliquer sur "Choisir parmi une liste de pilotes disponibles sur mon ordinateur et déselectionner l'option "Afficher les matériels compatibles". Une liste doit s'afficher, dans cette dernière chercher "ArduinoLLC" ou "Arduino Srl" puis chercher le modèle de votre carte Arduino et cliquer sur suivant.

Il se peut qu'un avertissement s'affiche sur votre ordinateur mais passer outre en appuyant sur le bouton "Oui".Une fois toutes ces étapes mener à bien il suffit d'attendre la mise à jour du pilote et pour terminer appuyer sur "Fermer". Un raffraichissement de la page du gestionnaires de périphériques devrait s'éffectuer. 

Pour vérifier si la mise à jour du pilote à fonctionner vérifier dans la section "Port (COM et LPT" de votre gestionnaire de périphériques. Si le composant porte maitenant le nom de votre carte Arduino alors l'opération est un succès et dans le logiciel TechVolt vous devriez voir apparaitre le numéro du port COM dans la section "Port" qui est associé à votre carte Arduino dans le gestionnaire de périphérique (COM X).
